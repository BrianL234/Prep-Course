Variables: una variable es un espacio que creamos a partir de la 
palabra clave "var" al cual le vamos a poner un nombre e ingresaremos
datos a ese espacio (mayormente en relación al nombre que le indicamos
a la variable) despues de un signo igual "=", esto se llama asignar.
 Se les llama variable porque pueden cambiar su valor. 

Strings: son cadenas de caracteres, conjunto de letras o palabras que
estan entre comillas. (Es una expresión primaria).

Funciones(argumentos, return): una función es un conjunto de líneas de
código que realizan una tarea específica y pueden retornar algo.
 Los argumentos son como variables que declaramos adentro de la función
que van a tomar un valor cuando se ejecute la misma.
 Una función return lo que hace es devolvernos un valor (no lo imprime)
 y finaliza la función.

Declaraciones "if": El control de flujo permite que nuestra función
verifique si algo es true o false e ir ejecutando diferentes codigos 
según ciertas condiciones, If es la palabra clave para ejecutarlo. 

Valores Booleanos (true, false): Los valores booleanos son un tipo de 
dato, solo puede tener dos tipos de valores, true o false.
